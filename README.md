What is Bitcoin Base?
-------------

Bitcoin Base (BCB) is a decentralized digital currency focused on user privacy, enabling secure and independent transactions. The mission is to become Peer-to-Peer Electronic Cash, capable of processing thousands of transactions per second (TPS) with very low fees and private payments to anyone. The management and operation are carried out entirely through the network free from any central authority or organizational.

Bitcoin Base Core is the name of the open source software which enables the use of this currency.


For more information read the original Bitcoin Base docs.
